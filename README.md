# Temperature Converter

A simple and easy-to-use tool that converts temperatures between Celsius, Fahrenheit, and Kelvin.

---

## Features

- Convert between **Celsius**, **Fahrenheit**, and **Kelvin**
- Instant, accurate results
- Clean and straightforward interface

---

## Temperature Scales

| Scale      | Abbreviation | Absolute Zero  |
|------------|--------------|----------------|
| Celsius    | °C           | −273.15 °C     |
| Fahrenheit | °F           | −459.67 °F     |
| Kelvin     | K            | 0 K            |

---

## Conversion Formulas

| From → To          | Formula                          |
|--------------------|----------------------------------|
| Celsius → Fahrenheit | °F = (°C × 9/5) + 32           |
| Fahrenheit → Celsius | °C = (°F − 32) × 5/9           |
| Celsius → Kelvin   | K = °C + 273.15                  |
| Kelvin → Celsius   | °C = K − 273.15                  |
| Fahrenheit → Kelvin | K = ((°F − 32) × 5/9) + 273.15  |
| Kelvin → Fahrenheit | °F = ((K − 273.15) × 9/5) + 32  |

---

## Usage

Enter a temperature value, select the source unit, and the converter will display the equivalent values in all other units.

### Examples

| Input         | Celsius | Fahrenheit | Kelvin  |
|---------------|---------|------------|---------|
| 0 °C          | 0       | 32         | 273.15  |
| 100 °C        | 100     | 212        | 373.15  |
| 37 °C (body)  | 37      | 98.6       | 310.15  |
| −40 °C        | −40     | −40        | 233.15  |
